# 🧮 Web Calculator

A simple and responsive web-based calculator built using **HTML, CSS, and JavaScript**.
This project performs basic arithmetic operations like addition, subtraction, multiplication, and division.

---

## 🚀 Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* 🧹 Clear (Reset) button
* 📱 Responsive design (works on mobile & desktop)

---

## 🛠️ Technologies Used

* HTML – Structure of the calculator
* CSS – Styling and layout
* JavaScript – Functionality and logic

---

## 📂 Project Structure

```
calculator/
│── index.html
│── style.css
│── script.js
```

---

## ▶️ How to Run

1. Download or clone the repository:

   ```
   git clone https://github.com/your-username/calculator.git
   ```

2. Open the project folder.

3. Double-click on `index.html`
   OR open it in your browser.

---

## 💡 How It Works

* Buttons take input from the user
* JavaScript processes the input
* Result is displayed on the screen
